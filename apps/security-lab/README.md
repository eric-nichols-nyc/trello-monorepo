# Security Lab

This folder groups the Security Lab client and server applications used for security experiments and demos.

## Structure

- `client/` – Next.js app for the Security Lab UI.
- `server/` – Express app that exposes the Security Lab API.

## Scripts

From this directory, you can run:

- `pnpm dev:client` – start the Next.js client.
- `pnpm dev:server` – start the Express server.
- `pnpm build:client` – build the client app for production.
- `pnpm start:client` – start the built client app.
- `pnpm start:server` – start the Express server in production mode.
